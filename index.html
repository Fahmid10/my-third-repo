Classification of memory

A computer system contains a variety of memory devices to store instruction and data for its operation. In general, memory components of a computer system can be divided into three categories:

  • Internal processor memory (or built-in memory)
  • Main memory or primary memory
  • Secondary memory

    • Internal Processor Memory: Basically it comprises of a small set of high-speed register used as a working memory for temporary storage of instruction and data. Some microprocessor also employ another type of built-in memory called cache.

    Main memory: It is relatively a large memory. This memory is used for program and data storage during computer operation. Location in memory can be accessed directly an rapidly by the CPU.

    Secondary Memory: This is generally much larger in capacity but is slower than main memory. It is used for storing system programs and large dat file, which are not continually required by the CPU.

    General Properties of Memory Devices

    Memories are based on a number of physical phenomenon and principles. Knowledge of these properties is essential in evaluating memory technologies.
